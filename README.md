# Fyxlife - Take Home Assignment (Expo + TypeScript)

This repository is a minimal Expo React Native app implementing the Fyxlife take-home assignment.

## Features included
- Onboarding flow (3 screens) - collects user profile and stores in AsyncStorage
- Main Dashboard with goal cards (Move, Eat, Calm)
- Progress screen (today / week / month summary)
- Risk-o-meter screen with grouped risks
- Local persistence via AsyncStorage
- Swap goal functionality
- Streak counter example
- Simple animations with `react-native-reanimated` (optional dependency)

## Tech stack
- Expo (managed)
- React Native + TypeScript
- React Navigation
- @react-native-async-storage/async-storage
- react-native-reanimated (for small animations)

## How to run
1. Install Expo CLI if not already: `npm install -g expo-cli`
2. Clone the repo or copy files.
3. `npm install` or `yarn`
4. `expo start` and open in Expo Go or simulator.
